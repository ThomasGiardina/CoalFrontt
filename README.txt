Pasos a Seguir para iniciar la app:

1-Para instalar las dependencias introducir en la terminal el comando:
    
    npm install
    
2-Luego para correr el codigo introducir en la terminal el comando

    npm run dev

3-Poner en el navegador:

    http://localhost:5173/


Comentarios adicionales:
    -El Home.html y el Store.html son ideas previas, luego de terminar el proyecto se eliminaran
